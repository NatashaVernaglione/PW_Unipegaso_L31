# PW_Unipegaso_L31
Progetto finale per il conseguimento della Laurea Triennale in Informatica per le aziende digitali (L-31) presso l'Università Telematica Pegaso. Sviluppo e implementazione della traccia d'esame relativa Sviluppo Web. Realizzato con html, css, javascript

# utilizzo
realizzazione pagina web, dove oltre a una breve introduzione su chi siano i fonatori, e la storia dell'azienda Planet Farms, si mette in evidenza quali sono i pilastri dell'azienda e i risultati con cui vengono rispettati i parametri ESG previsti dall'Unione Europea

# tecnologie utilizzate
è styato utilizzato un linguaggio di markup per quanto riguarda la struttura della pagina web, quindi html5, planetfarms.html
per lo stile, è stato utilizzato un foglio esterno, style.css
per l'animazione del counter riguardo la sezione della inclusività, è stato prodotto anche qui un foglio esterno, con un linguaggio javascript, script.js
per il report da scaricare, si è utilizzato il report originale, disponibile nella pagina ufficiale della planetfarms, PLANET FARMS_REPORT_FULL_11032024.pdf (il nome del file, è stato lasciato quello originale)
